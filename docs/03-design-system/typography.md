# Typography

## Общий подход

Типографика ЦОС «Пифагор» должна быть спокойной, читаемой и современной.

---

## Основные шрифты

Рекомендуемая связка:

- **Manrope** — основной интерфейсный шрифт;
- **Inter** — альтернативный интерфейсный шрифт;
- **Forum** — декоративный шрифт для отдельных брендовых элементов;
- **PT Sans** — дополнительный текстовый шрифт для публичных страниц.

---

## Иерархия

```css
h1 {
    font-size: 40px;
    line-height: 1.15;
    font-weight: 700;
}
h2 {
    font-size: 28px;
    line-height: 1.25;
    font-weight: 700;
}
h3 {
    font-size: 20px;
    line-height: 1.3;
    font-weight: 600;
}
body {
    font-size: 16px;
    line-height: 1.6;
    font-weight: 400;
}
small {
    font-size: 14px;
    line-height: 1.45;
    font-weight: 400;
}
caption {
    font-size: 12px;
    line-height: 1.4;
    font-weight: 500;
}
```

---

## Тон текстов

Писать нужно понятно, спокойно, без давления, без лишней официальности и с уважением к пользователю.

---

## Возрастная адаптация

Младшие классы: проще, теплее, короче. Средние классы: больше самостоятельности. Старшие классы: академичнее и конкретнее.
