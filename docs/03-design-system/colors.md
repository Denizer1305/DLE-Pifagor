# Colors

## Общая палитра

Цветовая система Пифагора строится вокруг спокойной образовательной палитры: глубокий сине-серый, мягкий светлый фон, белые поверхности, пастельные акценты и аккуратные статусы.

---

## Базовые цвета

```css
:root {
    --color-primary: #394458;
    --color-primary-dark: #273142;
    --color-primary-light: #EEF1F6;
    --color-background: #F6F7FA;
    --color-surface: #FFFFFF;
    --color-surface-muted: #F0F3F8;
    --color-text: #263142;
    --color-text-muted: #7A8497;
    --color-border: #E1E6EF;
}
```

---

## Акценты

```css
:root {
    --color-accent-blue: #4A6FA5;
    --color-accent-yellow: #F2C94C;
    --color-accent-coral: #E9857A;
    --color-accent-green: #6ABF8A;
    --color-accent-purple: #8E7CC3;
}
```

---

## Статусы

```css
:root {
    --color-success: #2F9E64;
    --color-warning: #D99A2B;
    --color-danger: #D9534F;
    --color-info: #3F7FBF;
}
```

---

## Правила

- основной текст должен быть контрастным;
- светло-серый текст нельзя использовать для важной информации;
- красный использовать только для ошибок и критичных статусов;
- жёлтый использовать для дедлайнов и мягких предупреждений;
- зелёный использовать для успеха;
- синий использовать для основных действий.
