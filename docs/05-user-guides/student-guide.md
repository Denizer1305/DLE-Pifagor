# Инструкция студента

## Назначение кабинета

Личный кабинет студента показывает учебное пространство, ближайшее расписание, задания, активность, календарь, заметки и уведомления.

## Основные действия

- просматривать ближайшее расписание;
- смотреть учебные события;
- вести заметки;
- получать уведомления;
- редактировать профиль и настройки.

## Пустые состояния

Если у студента пока нет данных, карточки должны показывать сообщение о том, что данные еще не добавлены.
