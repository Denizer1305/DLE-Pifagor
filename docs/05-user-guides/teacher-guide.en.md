# Teacher Guide

## Purpose

The teacher dashboard shows educational events, upcoming plan, notifications, notes, and work indicators.

## Main actions

- view the dashboard home page;
- work with calendar and notes;
- receive notifications;
- update profile;
- configure appearance and notifications.

## Empty states

If data has not been added yet, the interface should show a clear empty state instead of test data.
