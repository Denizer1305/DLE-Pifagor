# Инструкция преподавателя

## Назначение кабинета

Личный кабинет преподавателя показывает учебные события, ближайший план, уведомления, заметки и рабочие показатели.

## Основные действия

- просматривать главную страницу кабинета;
- работать с календарем и заметками;
- получать уведомления;
- обновлять профиль;
- настраивать внешний вид и уведомления.

## Пустые состояния

Если данные еще не добавлены, интерфейс должен показывать понятное пустое состояние вместо тестовых данных.
