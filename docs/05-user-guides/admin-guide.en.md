# Administrator Guide

## Purpose

The administrator dashboard is used to manage users, organizations, feedback requests, notifications, calendar events, notes, and platform settings.

## Main sections

- Home - platform summary and upcoming plan.
- Users - list, filters, creation, detail view, and editing.
- Organizations - educational organization cards and administrative actions.
- Feedback - user requests, statuses, attachments, and detailed view.
- Calendar - events and day plan.
- Notes - personal notes linked to dates.
- Notifications - system events and user signals.
- Settings - appearance, notifications, privacy, roles, and security.

## Users

Use filters and the users table. Open a user card or edit page to update data. Recovery email, phone, profile, and roles should be saved through backend API.

## Feedback requests

Open the detailed view to see full text, contacts, organization, and attached files. After opening a request notification, it may be marked as read.

## Organizations

The organizations module stores educational organization data. Academic structure should live in a separate structure module.
