# Инструкция родителя

## Назначение кабинета

Личный кабинет родителя предназначен для просмотра связанных учебных данных, уведомлений, календаря, заметок и профиля.

## Основные действия

- просматривать главную страницу кабинета;
- отслеживать уведомления;
- работать с календарем и заметками;
- обновлять профиль;
- настраивать отображение и уведомления.

## Связанные аккаунты

Родительские сценарии зависят от связанных аккаунтов и данных, возвращаемых backend.
