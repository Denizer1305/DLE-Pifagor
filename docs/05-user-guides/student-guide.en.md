# Student Guide

## Purpose

The student dashboard shows the learning workspace, upcoming schedule, assignments, activity, calendar, notes, and notifications.

## Main actions

- view upcoming schedule;
- check educational events;
- keep notes;
- receive notifications;
- edit profile and settings.

## Empty states

If the student has no data yet, cards should show that data has not been added yet.
