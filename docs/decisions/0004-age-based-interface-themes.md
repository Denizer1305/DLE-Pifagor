# ADR 0004: Age-Based Interface Themes

## Status

Accepted

## Context

Одна из ключевых идей ЦОС «Пифагор» — среда взрослеет вместе с учеником.

Интерфейс для младшего школьника не должен быть таким же, как интерфейс для старшеклассника или студента.

---

## Decision

Ввести возрастные темы интерфейса:

```text
junior
middle
senior
college
university
```

---

## Reasons

Возрастные темы помогают сделать интерфейс понятнее, повысить вовлечённость, сохранить единый бренд, адаптировать сложность интерфейса и поддержать идею роста ученика внутри Пифагора.

---

## Consequences

Плюсы: сильная продуктовая идея, отличимость от аналогов, лучшее соответствие возрасту пользователя.

Минусы: больше работы над дизайном и поддержкой нескольких визуальных тем.

---

## Rule

Возрастная тема может менять подачу, но не должна ломать структуру платформы и узнаваемость бренда.
