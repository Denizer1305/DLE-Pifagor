# Presentation Exports

Папка `exports/` предназначена для финальных и промежуточных экспортов презентаций ЦОС «Пифагор».

---

## Что можно хранить здесь

- PDF-версии презентаций;
- финальные `.pptx` для отправки;
- изображения отдельных слайдов;
- демо-версии;
- архивные версии;
- презентации для разных аудиторий.

---

## Рекомендуемая структура

```text
exports/
 ┣ pdf/
 ┣ pptx/
 ┣ images/
 ┗ archive/
```

---

## Примеры имён файлов

```text
pifagor-product-presentation-2026-05.pdf
pifagor-demo-for-schools-2026-05.pptx
pifagor-roadmap-slides-2026-05.pdf
```

---

## Правила

- не хранить временные файлы;
- не использовать названия вида `final_final_2.pptx`;
- указывать дату или версию;
- финальные версии сохранять отдельно от черновиков;
- большие архивные файлы не хранить в Git без необходимости.

---

## Главный принцип

> В `exports/` должны лежать только осмысленные результаты экспорта, которые можно открыть, показать или отправить.
