# Презентационный комплект ЦОС «Пифагор»

Папка `design/presentations/` хранит комплект для оформления презентаций в стиле ЦОС «Пифагор».

Этот комплект нужен для:

- защиты проекта;
- демонстрации продукта;
- презентаций для образовательных организаций;
- методических материалов;
- публичных выступлений;
- внутренних проектных документов.

---

## Структура

```text
presentations/
 ┣ exports/
 ┣ fonts/
 ┣ guides/
 ┣ templates/
 ┗ README.md
```

---

## `templates/`

```text
templates/
 ┗ pifagor-presentation-template.pptx
```

Основной шаблон презентации.

Рекомендуемое использование:

- создавать копию шаблона перед редактированием;
- не изменять базовые стили без необходимости;
- сохранять единую сетку слайдов;
- использовать фирменные цвета;
- не перегружать слайды текстом.

---

## `guides/`

```text
guides/
 ┗ pifagor-presentation-guide.pdf
```

Руководство по оформлению презентации.

В руководстве должны быть описаны:

- структура презентации;
- правила использования слайдов;
- типографика;
- цветовая палитра;
- работа с графиками;
- работа с таблицами;
- рекомендации по изображениям;
- финальная проверка перед показом.

---

## `fonts/`

```text
fonts/
 ┣ Forum-Regular.ttf
 ┣ PT_Sans-Web-Bold.ttf
 ┣ PT_Sans-Web-BoldItalic.ttf
 ┣ PT_Sans-Web-Italic.ttf
 ┗ PT_Sans-Web-Regular.ttf
```

Шрифты, используемые в презентационном шаблоне.

Важно:

> Не распространять шрифты отдельно без проверки лицензии. Если комплект будет передаваться пользователям, добавить отдельный файл `LICENSE.md` или пояснение по правам использования.

---

## `exports/`

Папка для экспортированных презентаций.

Сюда можно складывать:

- PDF-версии;
- финальные презентации;
- демо-экспорты;
- изображения отдельных слайдов;
- архивные версии.

---

## Рекомендации по оформлению

### Один слайд — одна мысль

Не перегружать слайд большим количеством текста.

### Визуальная иерархия

На слайде должен быть главный акцент:

- заголовок;
- ключевая мысль;
- график;
- схема;
- изображение.

### Спокойная палитра

Использовать фирменные цвета Пифагора:

- глубокий сине-серый;
- светлый фон;
- белые поверхности;
- мягкие акценты.

### Единый стиль

Не смешивать случайные шрифты, цвета и декоративные элементы.

### Проверка перед показом

Перед отправкой или выступлением проверить:

- читаемость текста;
- наличие шрифтов;
- корректность изображений;
- качество графиков;
- отсутствие лишних слайдов;
- правильность титульного и финального слайда.

---

## Когда использовать

Презентационный комплект подходит для:

- pitch-презентации проекта;
- защиты диплома;
- встречи с образовательной организацией;
- демонстрации модулей;
- описания дорожной карты;
- брендовых материалов.

---

## Главный принцип

> Презентация Пифагора должна выглядеть спокойно, уверенно и профессионально, не теряя человеческого и вдохновляющего характера проекта.
